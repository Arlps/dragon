《标线》

为 AmSerialChart, AmXYChart 和AmRadarChart 图表创建水平/垂直准线 - /区域，如果没有指定，则自动调整轴的设置。

Example

var chart = AmCharts.makeChart("chartdiv",{
  ...
  "guides": [
    {
      "fillAlpha": 0.10,
      "value": 0,
      "toValue": 10
    }
  ]
});
