Creates the balloons ( tooltips ) of the chart, It follows the mouse cursor when you roll-over the data items.
The framework generates the instances automatically you only need to adjust the appearance to your needs.

Example

var chart = AmCharts.makeChart("chartdiv", {
  ...
  "balloon": {
    "adjustBorderColor": true,
    "color": "#000000",
    "cornerRadius": 5,
    "fillColor": "#FFFFFF"
  }
});

Properties

Property Type Default Description
adjustBorderColor Boolean true 如果设置为true，当用户滚动slice, graph,等时，边框颜色而不是背景颜色将会更改。
在线实例:demo
animationDuration Number 0.3 从上一点到当前点的balloon运动持续时间，以秒为单位。
在线实例:demo
borderAlpha Number 1 toolTip边框透明度
在线实例:demo
borderColor Color #FFFFFF 气球边框颜色。 只y有adjustBorderColor为false时生效。
在线实例:demo
borderThickness Number 2 Balloon 边框厚度
在线实例:demo
color Color #000000 toolTip提示框（balloon）字体颜色
在线实例:demo
cornerRadius Number 0 Balloon 圆角半径
在线实例:demo
disableMouseEvents Boolean true 如果您的balloon具有链接，您必须将其设置为false，才能使这些链接可点击。
在线实例:demo
drop Boolean false 允许具有液滴形 balloons.请注意，这些balloons 不会检查与其他balloons的重叠 ,也不会检查它们是否在绘图区域外。 它也不会像常规balloons一样根据其垂直位置自动更改指针方向。 你可以使用pointerOrientation属性，如果你想要指向不同的方向。 IE8不支持。
在线实例：demo
enabled Boolean true 使用此属性可为某些值轴禁用balloons。 也就是说：:

		"valueAxes": [{
			  // ...
			  // value balloons are shown
			}, {
			  // ...
			  "balloon": {
				"enabled": false
			  }
			  // value balloons are not shown
			}]

在线实例：demo
fadeOutDuration Number 0.3 淡出动画的持续时间（以秒为单位）。
在线实例：demo
fillAlpha Number 0.8 Balloon 背景透明度。
在线实例：demo
fillColor Color #FFFFFF Balloon 背景颜色。 通常Balloon背景颜色由图表设置。 仅当“adjustBorderColor”为“true”时，才使用此颜色。
在线实例：demo
fixedPosition Boolean true 指定当悬停slice/column/bullet 时气球应跟随鼠标， 或保持在固定位置（如果使用ChartCursor ， 则不会影响balloon行为）。
在线实例：demo
fontSize Number balloon中文本的大小。 默认情况下使用图表的fontSize。
在线实例：demo
horizontalPadding Number 8 balloon的水平填充。
在线实例：demo
maxWidth Number balloon的最大宽度。
在线实例：demo
offsetX Number 1 定义从鼠标指针到balloon指针的水平距离。 如果将它设置为一个小值，气球可能会闪烁， 因为鼠标可能会对悬浮对象失去焦点。

注意：除非fixedPosition设置为false，否则将忽略此设置。
在线实例：demo
offsetY Number 6 定义从鼠标指针到balloon指针的垂直距离。 如果将它设置为一个小值，气球可能会闪烁， 因为鼠标可能会对悬浮对象失去焦点。

注意：除非fixedPosition设置为false，否则将忽略此设置。
在线实例：demo
pointerOrientation String down 只有当balloon.drop设置为true，指定指针的方向时才有效。
在线实例：demo
pointerWidth Number 6 指针的宽度（箭头）“root”。 仅当cornerRadius为0时使用。
在线实例：demo
shadowAlpha Number 0.4 阴影的透明度。
在线实例：demo
shadowColor Color #000000 阴影的颜色。
在线实例：demo
showBullet Boolean false 如果balloons的cornerRadius> 0，则在使用ChartCursor时， balloons 的showBullet设置为true。 如果不想在balloons 附近显示子弹， 请将其设置为false：chart.balloon.showBullet = false
在线实例：demo
textAlign String middle 文本对齐，可能的值:"left", "middle" and "right"
在线实例：demo
verticalPadding Number 4 balloon的垂直填充。
在线实例：demo
